Pokemon Colosseum Simulator

Author: Brandon Westbrook, N01156261

Files:"pokemon-data.csv", "move-data.csv", "PokemonColosseum.py", and "Pokemon.py"

Imports: namedtuple and random

This project simulates a Pokemon battle between two teams, the player's team and Team Rocket.
Each team is picked at random from the data provided in the two csv files.

To run:
	1. Provided files are within single directory
	2. Python version 3.11
	3. Run as "py PokemonColosseum.py"
	4. Follow prompts on screen to play through the pokemon battle
	5. Game will auto-close when either team's HP all reach 0 (faint)
